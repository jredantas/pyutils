Python library with some useful generic functions.

To install, git clone this repo and execute "pip install ." from repo path.

Available functions:

- Misc
    - create dir if not exists
    - dot product
    - lenght
    - angle
- Db
    - open db connection
    - close db connection
- Timer
    - tic
    - toc
    - get start date
    - set start date
- Logs
    - setup logger
    - setup main log
    - close logs
    - log all
